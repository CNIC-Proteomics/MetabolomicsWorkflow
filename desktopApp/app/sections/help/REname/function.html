What is REname?

REname (Regular Expressions processing of compound names) parses the name of the candidate metabolites to eliminate the information related to the isomer (or convert all the acid forms into salt), and combine the rows of the table with isomeric metabolites assigned to the same "feature". The processing of the name of the compounds is done using the Goslin library in the case of lipids, and a system of regular expressions for the rest of metabolites.

RowMerger
Function Settings

RowMerger receives an input table, in xls or xlsx format, and outputs a processed matrix with the same structure. The user must specify which properties of the annotations will be taken into account for the merge to take place (i.e. ComparedColumns). Likewise, it must be indicated which properties will be kept in the entry resulting from the merging of the annotations (i.e. ConservedColumns).
Parameters
  • ComparedColumns: Name of the columns whose values will be compared when merging two rows. That is, two rows will be merged if they have the same values in these columns.
  • ConservedColumns: Name of the columns whose values will be preserved in the row resulting from the merge. The values of the merged rows will appear separated by " // ".
  • OutputName: Name of the output file. By default, the name of the output file will be that of the input file with prefix "RowMerged_"
  • OutputColumns: Name of the columns present in the output table. By default, they will all appear.

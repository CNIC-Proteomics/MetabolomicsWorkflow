Tagger

Tagger is a classifier capable of detecting metabolites characterized as nutrients, drugs, microbial, halogenated or peptides. The main advantage of this tool consists in the possibility of classifying, from the list of possible identifications, those annotations that belong to any of the categories mentioned above. While the classification of halogenated and peptide compounds is carried out through the use of regular expressions, which offer great precision and speed, in the case of nutrients, drugs and microbial origin compounds, predefined lists updated periodically are used. Here are the details for each of the possible annotations:
  • Nutrients: The list of nutrients used by Tagger was built using the HMDB database, which includes more than 40,000 endogenous and exogenous metabolites detected in humans. Using its structured ontology, the compounds characterized as "Food" and "Food and Nutrition" were extracted, excluding metabolites whose origin could be endogenous ("Endogenous"). Given that the same metabolite can be named in different ways, it was necessary to "enrich" the list of metabolites with their synonyms, for which the PubChem database was used. It is possible to download the file with the list of nutrients from the repository on GitHub.
  • Drugs: The list of drugs used by Tagger was built using the HMDB database was used, which thanks to its structured ontology allows the extraction of compounds characterized as "Drug" or "Pharmaceutical industry", excluding compounds whose origin could be endogenous ("Endogenous”). In addition, the PubChem database was used to "enrich" the list of metabolites with their synonyms. It is possible to download the file with the list of drugs from the repository on GitHub.
  • Microbial Compounds: An in-house list was used for the identification of microbial metabolites, using PubChem database to include the synonyms of the different compounds. It is possible to download the file with the list of microbial compounds from the repository on GitHub.
  • Halogenated Compounds: Regular expressions are used for the identification of the halogenated compounds. Regular expressions define patterns with which to find a certain combination of characters in a text string. In this case, Tagger uses a regular expression that, applied to the name of a metabolite or its chemical formula, makes it possible to determine the presence of a halogen.

Regular Expression ([Ff]luor(?!ene)|[Cc]hlor(?!ophyl)|[Bb]rom|[Ii]od)
Compound 1 14,14,14-Trifluoro-11E-tetradecenyl acetate
Compound 2 6-(2-Chloroallylthio)purine
Compound 3 Bromhexine
Compound 4 Fluorene
Compound 5 Chlorophyl

  • Peptide compounds: Peptide compounds are detected through regular expressions, which recognize the amino acids in the compound's name. For this, the amino acids must be expressed in the three-letter nomenclature.

Regular Expression (?i)^(Ala|Arg|Asn|Asp|Cys|Gln|Glu|Gly|His|Ile|Leu|Lys|Met|Phe|Pro|Ser|Thr|Trp|Tyr|Val|[-,\s]){3,}$
Compound 1 Pro Gly
Compound 2 Glu Ala
Compound 3 Gly Leu Cys
Compound 4 Met-Ala-Ala
Compound 5 Ile,Gly,Cys


It should be noted that the user can modify some parameters before starting the execution of the module. Thus, for example, depending on the experimental conditions (use of drugs or halogenated compounds, among other options), the user can select which classifications he wishes to perform (all or a selection of some of them) and which regular expressions to use to identify the halogenated and peptide compounds. The only thing that the user has to enter is a table, in xls or xlsx format, which must contain at least one column named “Name” and, optionally, another one named “Chemical Formula”, where the name of the metabolites and its chemical formula are found. After execution, Tagger will generate a table equal to the one entered, with an additional column for each classification performed.

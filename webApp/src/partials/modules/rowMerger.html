RowMerger

RowMerger is an entity comparer that allows combining information from different annotations according to criteria determined by the user. Therefore, this module allows grouping annotations assigned to the same feature in a single entry in the table with the possible identifications, thus facilitating the visualization of the data for the biological interpretation of the results.

To execute RowMerger, the user must specify which properties of the annotations (i.e. table columns) will be taken into account for the merge to take place. Likewise, it must be indicated which properties will be kept in the entry resulting from the merging of the annotations. By default, RowMerger will combine annotations that have the same experimental mass (Experimental mass), adduct (Adduct), error (mz Error (ppm)) and molecular weight (Molecular Weight), in such a way that the resulting entry will retain the identifier (Identifier) ​​and the name (Name) of all merged annotations.

RowMerger receives an input table, in xls or xlsx format, and outputs a processed matrix with the same structure.
